# Sources et inspirations

AFERTES Connect est une application open source inspirée par les projets suivants. Nous remercions leurs auteurs pour leur contribution à la communauté open source.

---

## Nextcloud
- **URL**: https://github.com/nextcloud
- **Licence**: AGPL-3.0
- **Description**: Plateforme de collaboration et de partage de fichiers auto-hébergée
- **Fonctionnalités inspirées**:
  - Espace de stockage personnel (Mon Drive)
  - Gestion des fichiers et dossiers
  - Interface de navigation par breadcrumb
  - Quotas de stockage utilisateur
  - Partage de fichiers

---

## Papillon App
- **URL**: https://github.com/PapillonApp/Papillon
- **Licence**: GPL-3.0
- **Description**: Application mobile open source pour la vie scolaire
- **Fonctionnalités inspirées**:
  - Affichage des emplois du temps
  - Gestion des notes et moyennes
  - Interface de navigation par domaines de compétences (DC)
  - Design mobile-first
  - Système de notifications

---

## FOSSNote
- **URL**: https://github.com/xFufly/fossnote
- **Licence**: GPL-3.0
- **Description**: Serveur alternatif open source pour la gestion scolaire
- **Fonctionnalités inspirées**:
  - Architecture client-serveur
  - Gestion des données utilisateurs
  - Système d'authentification
  - Structure de données pour les notes et emplois du temps
